# BUSINESS CONTEXT - Phase 2: Cross-Market Substitution Coefficients

> **Версія:** 1.0 | **Створено:** 01.02.2026

---

## КЛЮЧОВІ ДОСЛІДНИЦЬКІ ПИТАННЯ

### 1. Загальна тенденція субститованості
> Чи мають досліджувані препарати загальну тенденцію субститованості
> не на 1 локальному ринку, а на ВСІХ досліджуваних локальних ринках?

**Контекст:** Phase 1 дає SHARE_INTERNAL для кожного препарату на кожному ринку окремо. Питання: чи є ці значення консистентними across markets?

### 2. Можливість визначення коефіцієнта
> Чи можна визначити коефіцієнт субститованості препарату,
> виходячи з даних його субститованості з різних локальних ринків?

**Контекст:** Якщо препарат має схожий SHARE_INTERNAL на різних ринках — можна визначити загальний коефіцієнт. Якщо значення сильно варіюються — коефіцієнт ненадійний.

### 3. Методологія розрахунку
> Як правильно розрахувати такий коефіцієнт субститованості?

**Рішення:** Зважене середнє (вага = INTERNAL_LIFT) + довірчий інтервал для оцінки невизначеності.

### 4. Статистичні вимоги
> Яким статистичним вимогам повинен відповідати препарат
> для надійного визначення коефіцієнта субститованості?

**Вимоги:**
- Мінімальне покриття ринків (≥ 10%)
- Прийнятний коефіцієнт варіації (CV < 30% для високої надійності)

## БІЗНЕС-ЦІННІСТЬ PHASE 2

### Проблема
Phase 1 дає результати **per market**, але бізнесу потрібні **загальні** рекомендації:
- Чи можна оптимізувати SKU для препарату X **по всій мережі**?
- Чи є препарат Y критичним **для всіх аптек**?

### Рішення
Phase 2 агрегує дані з 100+ ринків для визначення **стабільних** патернів субституції.

```
Приклад:
Препарат X на 85 ринках:
├─ SHARE_INTERNAL: 72% ± 5% (CV = 12%)
├─ Висновок: СТАБІЛЬНА субституція
└─ Рекомендація: можна оптимізувати SKU по всій мережі

Препарат Y на 60 ринках:
├─ SHARE_INTERNAL: 45% ± 25% (CV = 55%)
├─ Висновок: НЕСТАБІЛЬНА субституція
└─ Рекомендація: аналізувати індивідуально по регіонах
```

## МЕТРИКИ ТА ІНТЕРПРЕТАЦІЯ

### Основні метрики Phase 2

| Метрика | Формула | Інтерпретація |
|---------|---------|---------------|
| **WEIGHTED_MEAN_SHARE** | Σ(SHARE × LIFT) / Σ(LIFT) | Загальний коефіцієнт субституції |
| **CI_95** | MEAN ± 1.96 × (STD/√N) | Діапазон "справжнього" коефіцієнта |
| **CV_PERCENT** | (STD/MEAN) × 100 | Стабільність across markets |
| **COVERAGE_PERCENT** | N_markets / TOTAL × 100 | Репрезентативність даних |

### Як читати результати

```
WEIGHTED_MEAN_SHARE = 72% означає:
  → В середньому 72% попиту залишається в аптеці при stock-out

CI_95 = [68%, 76%] означає:
  → "Справжній" коефіцієнт з 95% впевненістю в цьому діапазоні

CV = 12% означає:
  → Низька варіативність — коефіцієнт стабільний across markets

COVERAGE = 85% означає:
  → Препарат досліджений на 85% ринків — висока репрезентативність
```

## КЛАСТЕРИЗАЦІЯ ПРЕПАРАТІВ

### За Coverage (гібридний підхід)

| Кластер | Coverage | Статистична значущість |
|---------|----------|------------------------|
| **HIGH** | ≥ 50% | Висока — достатньо для надійних тестів |
| **MEDIUM** | 20-49% | Достатня — базові висновки можливі |
| **LOW** | 10-19% | Обмежена — обережність у висновках |
| **INSUFFICIENT** | < 10% | Недостатня — окремий датасет |

### За Reliability (CV)

| Reliability | CV | Інтерпретація |
|-------------|-----|---------------|
| **HIGH** | < 15% | Стабільна субституція across markets |
| **MEDIUM** | 15-30% | Помірна варіативність |
| **LOW** | > 30% | Нестабільна — коефіцієнт ненадійний |

## ОБМЕЖЕННЯ PHASE 2

### Препарати без даних Phase 1

**Проблема:** Phase 1 дає коефіцієнти тільки для препаратів зі stock-out подіями. Решта препаратів не мають даних.

**Рішення:** Ці препарати не включаються в Phase 2 аналіз. Для них потрібне окреме дослідження.

### Препарати з низьким coverage

**Проблема:** Препарати, присутні на < 10% ринків, не мають достатньо даних для надійних висновків.

**Рішення:** Виносяться в окремий датасет (INSUFFICIENT) для подальшого аналізу або очікування додаткових даних.

## БІЗНЕС-РЕКОМЕНДАЦІЇ

### За результатами Phase 2

| Профіль препарату | Характеристика | Рекомендація |
|-------------------|----------------|--------------|
| HIGH Coverage + HIGH Reliability | Стабільний across markets | Централізовані рішення по мережі |
| HIGH Coverage + LOW Reliability | Різна поведінка по ринках | Регіональна стратегія |
| LOW Coverage | Недостатньо даних | Накопичувати дані, відкладений аналіз |

## ГЛОСАРІЙ Phase 2

| Термін | Визначення |
|--------|------------|
| **Cross-Market Aggregation** | Об'єднання даних з різних локальних ринків |
| **Weighted Mean** | Зважене середнє (вага = INTERNAL_LIFT) |
| **Coverage** | Частка ринків, де препарат має дані |
| **Coefficient of Variation (CV)** | Відносне стандартне відхилення |
| **Confidence Interval (CI)** | Діапазон невизначеності оцінки |
| **Reliability** | Оцінка надійності коефіцієнта (на основі CV) |
